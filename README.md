# Genius Master Academy (GMA)

> **South Africa's Premium AI-Powered Academic Prep Platform for NSC Matric Students**

Genius Master Academy (GMA) is a high-yield, hyper-personalized, and beautifully designed digital learning platform engineered to solve the academic performance crisis in South Africa. Designed by top achievers, GMA combines premium NSC past papers, curated study guides, peer-to-peer co-study rooms, and advanced Gemini-powered AI tutoring into a single, high-fidelity experience.

### 🔍 Cost Breakdown & Feasibility (Gemini API vs. Pricing)

*   **Underlying Model**: Gemini 1.5 Flash (or Gemini 2.5 Flash-Lite).
*   **Token Pricing Basis**: $0.075 / million input tokens, $0.30 / million output tokens.
*   **Per-Question Token Cost**: 
    *   *Input Context* (Question metadata, partial paper context, user query) $\approx$ 2,000 tokens $\rightarrow$ $0.00015 USD
    *   *Output Reasoning* (Step-by-step solution, conceptual guidance) $\approx$ 500 tokens $\rightarrow$ $0.00015 USD
    *   *Total Cost per AI interaction* $\approx$ $0.0003 USD (approx. R0.005 ZAR).
*   **Result**: 30 questions cost only R0.16 per student per month, enabling an extremely lucrative, high-margin B2C business at a highly accessible R7.50 price point.

## 🛠️ Core Technology Stack & The Multidimensional Academic Ecosystem

GMA is engineered as a comprehensive learning ecosystem. It moves entirely away from simple chatbot bubbles to provide structured, multi-modal paths for every learning style.

```
                  ┌─────────────────────────────────────────┐
                  │      Genius Master Academy Hub          │
                  └─────────────────────────────────────────┘
                                       │
         ┌─────────────────────────────┼─────────────────────────────┐
         ▼                             ▼                             ▼
┌─────────────────┐           ┌─────────────────┐           ┌─────────────────┐
│ ACTIVE LEARNING │           │  AI DIAGNOSTICS │           │ SOCIAL & MOTIVATION│
├─────────────────┤           ├─────────────────┤           ├─────────────────┤
│ • Past Papers   │           │ • Smart Planner │           │ • Co-Study Rooms│
│ • Video Lessons │           │ • Weakness Packs│           │ • Streak Badges │
│ • Interactive Quizzes       │ • Progress Stats│           │ • Leaderboards  │
└─────────────────┘           └─────────────────┘           └─────────────────┘
```

### 1. The Active Learning Hub (Beyond the Chat)
*   **The Interactive Paper Engine**: Rather than just viewing static PDF exam papers, students interact with a multi-pane workspace. They can view questions, toggle official marking guidelines side-by-side, access an on-demand AI hint engine, or upload hand-written answers for automated visual marking.
*   **Integrated Video Lessons**: High-yield video content paired directly with NSC curriculum topics. Videos are augmented with:
    *   **Live Video Q&A**: An AI assistant synced with the video's transcript, letting students ask questions about what the instructor is saying *in real-time*.
    *   **Timestamped Quizzes**: Auto-generated interactive checkpoint quizzes that test comprehension at critical moments during playback.
*   **Curated Study & Exam Packs**: Expertly assembled offline-ready content bundles, past exam breakdowns, and high-yield study notes compiled by the country's top-performing matriculants.

### 2. Intelligent AI Diagnostics & Personalization
*   **The AI Smart Planner**: A real-time, personalized coordinator widget. Instead of a standard calendar, it queries the student's historical exam results, video watch times, and practice questions to auto-arrange high-efficiency, personalized weekly timetables.
*   **AI Weakness Pack Builder**: One-click generation of custom remedial packages. If a user struggles with "Quadratic Equations" on their last three papers, the system automatically runs deep prompts to compile a tailored workbook containing concept summaries, targeted drills, and step-by-step solutions for that specific weakness.
*   **Real-Time Exam Countdowns**: Beautifully synchronized countdown clocks linked directly to the official National Senior Certificate (NSC) exam timetable to keep student focus sharp and urgent.

### 3. Peer Accountability & Social Learning (The Anti-Isolation Layer)
*   **Moderated Live Co-Study Rooms**: Virtual study rooms where learners from across the country can gather, configure custom Pomodoro study timers, track group goals, and chat safely.
*   **Automated Content Moderation**: Advanced server-side filters block the sharing of personally identifiable information (PII), social handles, or toxic comments instantly to maintain an elite, distraction-free environment.
*   **Gamified Achievement Engine**: Integrated student leaderboards, profile levels, and streak systems tracking "Study Streaks" to incentivize daily active learning.

### 4. Enterprise-Grade Administration Dashboard
*   **Direct Content Ingestion**: Admin screens to manage past papers, ingest new PDF links, upload video masterclasses, organize metadata, and audit safety logs.
*   **Real-time Interaction Logs**: Full logging of AI tokens used, active server endpoints, and user-reported issues for rapid troubleshooting.

## ⚙️ How to Run & Configure

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Environment Variables
Create a `.env` file in the root directory:
```env
# Server Secrets (Server-side only, do not expose to client)
GEMINI_API_KEY=your_gemini_api_key_here

# Firebase Configuration
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

### 3. Start Development Server
```bash
npm run dev
```
This runs the full-stack server on `http://localhost:3000`.

### 4. Build for Production
```bash
npm run build
```
This builds the React static files to `dist/` and bundles `server.ts` into a self-contained production bundle at `dist/server.cjs` using `esbuild`.
